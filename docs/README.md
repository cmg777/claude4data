# Documentation README

Welcome to the documentation for the claude4data project. This README provides an overview of available documentation resources and how to navigate them.

## Available Reports

### RF Embeddings Comparison Report
- **File:** `rf_embeddings_comparison_report.html` (recommended for viewing)
- **Alternative formats:** `rf_embeddings_comparison_report.md`, `rf_embeddings_comparison_report.pdf`
- **Description:** Comprehensive analysis comparing regular vs. population-weighted satellite embeddings for predicting public service indicators across 339 Bolivian municipalities
- **Key content:**
  - Executive summary with statistical findings
  - Methodology and model specifications
  - Detailed results tables and visualizations
  - Interactive Mermaid.js diagrams
  - Discussion and policy implications
